    Awesome Invenio¶

    A curated list of awesome InvenioRDM and Invenio extensions, overlays and resources.

    Inspired by awesome-python.

    Overlays¶

    • CaltechDATA - The CaltechDATA InvenioRDM source code.
    • CaltechAUTHORS - The CaltechAUTHORS InvenioRDM source code.
    • CDS - New CERN Institutional Repository CDS, based on InvenioRDM.
    • UltraViolet - NYU Libraries' DLTS deployment of the InvenioRDM framework.
    • Zenodo - Zenodo.org source code, powered by InvenioRDM.

    Deployments¶

    • TU Graz Deployment - Docker compose based deployment.
    • Caltech Authors Single VM based deployment of v12b.

    Extensions¶

    Subjects¶

    • invenio-subjects-mesh - MeSH topical subject terms with qualifiers for InvenioRDM
    • invenio-subjects-mesh-lite - MeSH topical subject terms without qualifiers for InvenioRDM
    • invenio-subjects-lcsh - LCSH subject terms for InvenioRDM
    • galter-subjects-utils - Subjects tooling for InvenioRDM.
    • invenio-subjects-nasa - Nasa thesaurus subject terms for InvenioRDM.
    • invenio-subjects-cessda - CESSDA subject terms for Invenio RDM.
    • invenio-vocabularies-extra - Add some extras to the vocabularies module like jobs, readers and transformers for DDC, bilingual MeSH and GND subjects.

    Workflows¶

    • invenio-curations - Enables a curation workflow before publication of records.

    Persistent Identifiers¶

    • invenio-pidstore-extra - Adds support for URN minting for german and swiss libraries via API of the Deutsche Nationalbibliothek.

    Configuration¶

    • invenio-config-tuw - Customized workflows and baseline configuration for TU Wien Research Data.
    • invenio-config-tugraz - Override configuration from different Invenio modules to meet TU Graz requirements.

    Theming¶

    • invenio-theme-tuw - Corporate design for TU Wien, used in TU Wien Research Data.
    • invenio-theme-tugraz - TU Graz standard theme for InvenioRDM.

    Analytics/Statistics¶

    • invenio-analytics-importer - CLI to retrieve and ingest (Matomo) analytics into your InvenioRDM instance

    General¶

    • invenio-utilities-tuw - Extra CLI utilities for InvenioRDM, of varying quality and usefulness.

    For developers¶

    • InvenioRDM Devbox blueprint for a new take on developing
    • InvenioRDM Starter - A single docker image installation of InvenioRDM
    • Translation tools - check for inconsistencies, merge transifex into a single .po file, upload back, check with the help of chatgpt, apply newest translations to an installed RDM.
